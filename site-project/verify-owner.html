Verify Work Owner

This is done so that no one else can pass off the project as their own.

To verify work, you need:

  • 1. Copy the code from the message that I sent you with the work
  • 2. Insert the code in the 'Paste code here' field
  • 3. Click on 'Verify' button

Thats it! After that, you will receive an answer whether I am the owner of the website or not

Why is this safe?

  • The code is encrypted in sha256, which means that no one will be able to obtain the source code, even if they look at the page's source code
  • The code is issued only to the inspector, and to no one else. That is, no one other than the inspector will be able to verify the owner.
